Version 3.5 Push Professional

Neu in dieser Version:
- Push-Version im Code auf 3.5 angehoben
- Geräte können jetzt mit Namen registriert werden
- Push-Geräteübersicht im Kalender
- letzter erfolgreicher Push wird gespeichert
- Fehlerstatus und Fehlversuche pro Gerät werden gespeichert
- ungültige Geräte werden beim Senden automatisch entfernt
- Push-Test direkt aus der App

Nach dem Deploy:
1. Bei Render neu deployen
2. Auf dem Handy Seite neu laden
3. Push einmal neu aktivieren
4. Gerätenamen eintragen (z. B. Ute iPhone)
5. Im Kalender prüfen, ob das Gerät in der Liste erscheint
