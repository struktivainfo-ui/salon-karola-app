Wichtig für automatische E-Mails
1. In Render Environment Variablen setzen:
   SMTP_HOST
   SMTP_PORT
   SMTP_USERNAME
   SMTP_PASSWORD
   SMTP_SENDER
2. Optional:
   SMTP_USE_TLS=true
   SMTP_USE_SSL=false
3. Danach neu deployen
4. Im Dashboard 'Automatik prüfen' klicken
5. Bei Fehlern wird der letzte SMTP-Fehler jetzt direkt im Dashboard angezeigt

Hinweis
Ganz fehlerfrei kann die Automatik nur laufen, wenn SMTP-Daten korrekt gesetzt sind. Der Code ist in 3.3 robuster, aber ohne korrekte Mail-Zugangsdaten kann kein Versand erfolgen.
